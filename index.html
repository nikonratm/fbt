FBT: RSS Background Tabs Safari Extension

A Safari extension to create a shortcut button for opening feedly items in a new background tab. Formerly known as Feedly Background Tabs

Now re-built as a macOS app!

View the Project on GitHub nikonratm/fbt

For avid Feedly users keyboard shortcuts are a must but the default behavior is that a new tab immediately opens, taking over your browsing. This little extension fixes that, you can assign a shortcut key so that new tabs open in the background. Sounds simple but once you try it you'll never want to Feedly without it.

Originally inspired by http://nox.cx/zakj

Feedly Background Tabs Safari Extension

A Safari extension to create a shortcut button for opening feedly items in a new *background* tab

View the Project on GitHub nikonratm/feedlybackgroundtabs

  • Download Extension

  • View On GitHub

With the impending doom of Google Reader, this little extension is a clone of an old Safari extension that allows you to assign a shortcut key to open an item in a new tab—in the background.

Hopefully it makes the transition a bit easier for some other folks out there too.

Based on original work by http://nox.cx/zakj
